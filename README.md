# Учебный проект

## Описание:

Проект реализует функции для работы с банковскими данными 

## Функции:

### `filter_by_state(transactions, state='EXECUTED')`
Фильтрует список словарей по значению ключа 'state'.

**Параметры:**
- `transactions` (list[dict]): Список словарей с транзакциями
- `state` (str, optional): Значение для фильтрации. По умолчанию 'EXECUTED'

**Возвращает:**
- list[dict]: Список словарей, отфильтрованных по статусу

**Пример использования:**
```
data = [
    {'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'},
    {'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}
]

# Фильтрация выполненных операций
executed_ops = filter_by_state(data)
# Фильтрация отмененных операций
canceled_ops = filter_by_state(data, 'CANCELED')
```
